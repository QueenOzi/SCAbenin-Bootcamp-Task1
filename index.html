Java script Lesson 1

JAVASCRIPT HOME

JavaScript is easy language to learn. it is the programming language of HTML and the Web. Web pages are not the only place where JavaScript is used. Many desktop and server programs use JavaScript. Also Javascript and Java are completely different languages.

JAVASCRIPT INTRODUCTION

Javascript can do a lot of things. Examples include
1. Javascript can change HTML content: One of the method is getElementById().This method is used to "find" an HTML element (with id="demo") and changes the element content (innerHTML) to "Hello JavaScript"
2. It can change HTML attribute values
3. It can change HTML styles
4. It can show and hide HTML elements

JAVASCRIPT WHERE TO

JavaScript is the default scripting language in HTML. The javascript codes are inserted in between the script tags. You can place any number of scripts in an HTML document. Scripts can be placed in the body or in the head section of an HTML page, or in both. Scripts can be placed in external files. External scripts can be referenced with a full URL or with a path relative to the current web page.

JAVASCRIPT OUTPUT

Javascript can display data in different ways. This include:
1. Using innerHTML
2. Using document.write()
3. Uding window.alert()
4. Using console.log()

JAVASCRIPT STATEMENTS

JavaScript statements are composed of: Values, Operators, Expressions, Keywords, and Comments. The statements are executed, one by one, in the same order as they are written. JavaScript statements often start with a keyword to identify the JavaScript action to be performed.

JAVASCRIPT SYNTAX

JavaScripts syntax are sets of rules that shows how JavaScript programs are constructed. The JavaScript syntax defines two types of values: Fixed values and variable values. Fixed values are called literals. Variable values are called variables. JavaScript is case sensitive. It uses strings, operators, expression, keyword and identifers.

JAVASCRIPT COMMENTS

In JavaScript comments are used to give explain JavaScript code and to make it more readable. Single line comments start with //.
Multi-line comments start with /* and end with */. Any text between /* and */ will be ignored by JavaScript.

JAVASCRIPT VARIABLES

Variables are containers for storing data. All JavaScript variables must be identified with unique names. These unique names are called identifiers. Identifiers can be short names (like x and y) or more descriptive names (age, sum, totalVolume). The equal sign (=) is an "assignment" operatorin JavaScript not an "equal to" operator.

JAVASCRIPT OPERATORS

There are various types of operators. This include the assignment operator, increment operator, logical operator, expomentiation, string operators, comparison operators etc

JAVASCRIPT ARITHMENTIC

Arithmetic operators perform arithmetic on numbers (literals or variables). Operator precedence describes the order in which operations are performed in an arithmetic expression. Multiplication (*) and division (/) have higher precedence than addition (+) and subtraction (-). And the precedence can be changed by using parentheses.
